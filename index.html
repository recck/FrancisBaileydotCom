Uploadr

As a self taught programmer I'm always looking for ways to challenge myself. I was looking for a new project to work on and realized a file-management app would be really handy. That's when I decided to create Uploadr.

Using ajax, jQuery, and PHP Uploadr can upload and track the progress of multiple files in a single request. If an error occurs Uploadr continues uploading other files and returns useful error information to the user. If for some reason javascript is disabled, Uploadr will still function, but without progress tracking. You can check out the code on github or click the link below to view a live demo.

Aspen Global Systems

Aspen Global Systems is an accounting software resale company. They wanted to expand their presence online and hired me to create a new website for their business. I looked after the project from the ground up.

I started by setting up shared hosting and email service. Afterwards, I created the site using wordpress and a pre-made theme. Some parts of the theme were not up to par, so I added custom tweaks using css. I also added a few plugins to help bring functionality to the site. Thanks to my prior experience working with wordpress I was able to deliver the site on time and on budget.
